La **Raccomandazione TR 61-01** permette l'uso temporaneo delle stazioni radioamatore in paesi CEPT diversi dal proprio.

### Condizioni per l'Utilizzo Temporaneo

1. **Licenza valida**: Classe CEPT riconosciuta
